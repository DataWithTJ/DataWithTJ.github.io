Demand Forecast
Model

The aim of this project was to develop a reliable demand forecasting model for predicting revenue. I undertook extensive data preprocessing, including encoding categorical variables and scaling numerical features. Various models like ARIMA, Auto ARIMA, Random Forest, XGBoost, LSTM, and SVR were explored. The Random Forest model emerged as the most accurate, exhibiting low error metrics (MAE, RMSE, MAPE). Post-model analysis involved inverse scaling to align predictions with actual revenue figures. This project highlights my skills in machine learning, data preprocessing, and model evaluation, culminating in a robust predictive tool for strategic decision-making.
